logout
======

Выход из приложения.

Удаляется сессия на бэкенде. У клиента удаляется сессионная кука.

**URL**::

    /{v}/accounts/logout/

**Method**::

    POST

**Параметры запроса**

Входящих параметров нет.

**Структура data**

Пустой словарь.

**Возможные ошибки**

* INTERNAL_ERROR
* ACCOUNT_NOT_FOUND

**Пример ответа**

.. code-block:: javascript

    {
        "status": "ok",
        "data": {}
    }
